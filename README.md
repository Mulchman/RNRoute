# Raid Ninja Route
Add functionality to raid.ninja to jump between routes without having multiple tabs open. Can also use query parameters to set the 3 locations, eg. `?loc1=GAUNTLET&loc2=ROYAL_POOLS&loc3=PLEASURE_GARDENS`.
